# 秘書M Phase 1 — 要件定義書

## 1. プロダクト概要

### コンセプト
「言わなくても動く参謀」

単なる便利ツールではなく、昇悟さんの「参謀」として機能するAIエージェント。

### Phase 1のゴール
LINEでGmail・Calendar・タスクが操作できる状態にする。

## 2. 機能一覧

| 優先度 | 機能名 | 概要 |
|---|---|---|
| ★★★ | LINE連携 | LINEでの全操作・受信・返信 |
| ★★★ | Googleカレンダー | 予定確認・登録・変更 |
| ★★★ | Gmail管理 | 未読整理・要約・返信代行 |
| ★★★ | タスク管理 | 登録・優先順位付け・進捗管理 |
| ★★★ | 朝ブリーフィング | 毎朝7時の自動LINE送信 |
| ★★ | リマインド | 締切警告・重要タスク通知 |

## 3. 機能詳細

### 3-1. LINE連携
- LINEでメッセージを送るとClaudeが返答する
- 全機能のインターフェースはLINE経由

### 3-2. Googleカレンダー
- 「今日の予定は？」で当日の予定を返答
- 「明日14時に〇〇を追加して」で予定を登録
- 「明日の会議を15時に変更して」で予定を変更

### 3-3. Gmail管理
- 直近5日間の未読メールを重要度でフィルタリング
- 各メールを3行以内で要約してLINEに送信
- 「〇〇に返信して」でメール下書き作成→承認後送信

### 3-4. タスク管理
- 「〇〇をタスクに追加して」でSupabaseに保存
- 重要度×緊急度の2軸で自動分類（4象限）
- 「今1時間あるけど何やる？」で最適タスクを提示
- 期限が迫ったタスクを自動でLINE通知

### 3-5. 朝ブリーフィング
- 毎朝7時にLINEへ自動送信
- 内容：今日の予定・優先タスク・未読重要メール・警告事項
- 300文字以内・箇条書き

## 4. システム構成
LINE
↓
FastAPI（Railway）
↓
Claude API（claude-sonnet-4-6）
↓
Gmail API / Google Calendar API / Supabase
↓
LINEに返答
